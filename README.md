# EPUB Speedread (Flask)

Простое веб-приложение для конвертации EPUB. Форматирование:
- выделение полужирным первых слогов слов (русское слогоразделение — эвристически),
- красная строка в начале абзацев,
- отбивка абзацев через отступ снизу.

## Запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Открыть: http://127.0.0.1:5000

## Примечания
- Обработка идет только внутри `<body>` HTML/XHTML.
- Изображения, оглавление и прочие ресурсы сохраняются без изменений.
